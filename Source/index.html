Yunus Programming Language 
 (Yunus is the most intelligent creature of the ocean. Yunus means Dolphin in Turkish.)

Producer:Tarik Bagriyanik  tbagriyanik@mynet.com yunus.sourceforge.net 
Start Date: November 2001
Update:  September 2009
Type: Open Source Freeware Visual Programming Language
Total Lines: 24.000 (600 hours)
Aim: To produce an object oriented visual language by using : Delphi
Source: www.delphiturk.com, DelphiTurk KodBank, Delphi 3 Marco Cantu, my students (Ilyas Vanlioglu, Mehmet Suslu), Kepler Script Engine Version 1.0 Coded by Cosseno
State: All the programming commands are not finished. There are just some kind of simple statements and reaching object properties. We have some basic Turkish/English menu system, saving, printer output, compiling, colored code, automatic completing, XML database and Excel file usage. At every program you can customize Yunus' own menus. Error Window is not finished.

When Yunus Program is run, you can run *.proje files from Windows Explorer. Also you can drag and drop *.proje files to the main window to open.
It has a very simple EXE compiling (if you have Delphi6 or upper ) and HTML export. Delphi can open the produced *.dpr file.

TSynEdit component needs to be installed to Delphi, if you want to enhance this project.
ZLPortIO unit is used for physical parallel port usage for all Windows operating systems.
